# World Translator Documentation

## Project Overview
The World Translator project is designed to provide seamless translations between various languages, making communication easier and more effective.

## Features
- Real-time translation of text
- Support for multiple languages
- User-friendly interface
- Language detection

## Technologies Used
- Frontend: HTML, CSS, JavaScript
- Backend: Node.js, Express
- Database: MongoDB

## Getting Started
1. Clone the repository:
   ```bash
   git clone https://github.com/mohammadbarghouthi19-sys/World-Translator-
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the server:
   ```bash
   npm start
   ```
